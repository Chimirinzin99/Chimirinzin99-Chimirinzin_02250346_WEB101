# TikTok Clone Web Project
-----
Aim:
-----
This project is to replicate a  simple TikTok-style web application inspired by TikTok. It allows users to watch short videos in a scrollable feed, like videos, and view a basic user interface similar to modern social media platforms.

---------
Features
---------

- User login and registration system
- Vertical scrolling video feeds
- Heart button functionality
- Basic user profile section
- Explore Page

------------------
Technologies Used
------------------

- Frontend: React, Tailwind CSS
- Backend: Node.js, Express.js

-----------------------
How to Run the Project
-----------------------

1. Clone the repository
2. Cd if necessary
3. Install dependencies using:
   npm install
4. Start frontend:
   npm run dev
5. Open browser at:
   http://localhost:3000
